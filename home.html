Introducing PiNo

PiNo is part of the 'Pirro Nord: a GIS approach' project. PiNo is a Web-based recording system designed to be able to manage excavation data. Data from field reports are digitised and linked to the back-end database. PiNo includes Data Entry, Data Edit, Data View and Data Export main functions, all delivered using a Web-based front-end. PiNo also integrates spatial data recorded using the Total Station, but currently is not able to display spatial features.

Data Entry
The Data Entry system of PiNo is defined to closely mirror the paper record, making data entry as intuitive as possible.
Data Edit
Data Edit is usefull to update information about records which are just recorded in the database.
Data View
The Data View is used for viewing individual records (as entered on the Data Entry page) and their associated data. The record view combines and brings together all linked items and data types that are associated with a particular record into one view. This can involve photos, plan-map images, text and spatial data soon
Data Export
This page is designed to enable user apply basic and advanced search functions on the database, with the options of viewing and exporting the results in CSV format, usefull to been imported in other software for statistical analysis.


/!\ PiNo is still under development at this test stage and is not recommended for official work. /!\
